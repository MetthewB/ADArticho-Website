Models & Methods

Discover the methodologies and models used to analyze humor in our datasets.

1. Data Integration and Preprocessing

Goal: Create harmonized, comparable datasets for linguistic and semantic analysis.

  • Data loading and standardization: Align OHIC to match the NYCC schema (shared columns: caption, funniness_score, year).
  • Text Cleaning & Tokenization: Prepare a cleaned version of each caption while preserving the original text (lowercase, remove punctuation and stopwords).
  • Linguistic and stylometric analysis:
    • Word count and character count
    • Type-Token Ratio (TTR) for lexical diversity
    • Formality score: ratio of nouns/adjectives to pronouns/verbs
    • Sentiment score using NLTK, TextBlob, or VADER

2. Thematic NLP and Semantic Modeling

Goal 1: Identify recurring humor types with zero-shot classification.

  • Define humor categories (awkward, sarcastic, dark, etc.) by describing them and providing examples.
  • Generate Transformer-based embeddings for each caption and each humor category description.
  • Compare each caption to the humor categories embeddings (with cosine similarity).
  • Classify each caption into its closest humor category.

Goal 2: Identify topic themes with zero-shot classification.

  • Repeat the same method as for humor types, by defining topic themes (politics, love, food, work, school, etc.).

3. Comparative Statistical and Stylistic Analysis

Goal: Quantify differences between humor contexts.

  • Thematic Comparison: Use Chi²/ANOVA tests to compare theme frequencies.
  • Stylistic Comparison: Conduct t-tests on linguistic features (TTR, formality, sentiment).
  • Bubble Classification: Train a logistic regression model predicting a caption’s origin (NYCC vs. OHIC). Interpret coefficients to identify features that define the “New Yorker bubble.”

4. Temporal and Cultural Trend Analysis

Goal: Track how humor evolves across time.

  • Group captions by year and compute theme proportions.
  • Detect major shifts in humor trends.
  • Correlate with sociopolitical events using contextual data.
  • Visualize using stacked bar charts or rolling averages.
